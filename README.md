## Dupesnoop

Dupesnoop is a duplicate file finder. It helps you locate files in a directory that are exact copies of each other, optionally allowing you to delete those files to reclaim the wasted space.

### Installation

To install Dupesnoop navigate to the [latest release](https://github.com/jyecusch/dupesnoop/releases) page, then download the file appropriate for your operating system or choose one of the downloads below:

- [MacOS Installer Download](https://github.com/jyecusch/dupesnoop/releases/latest/download/dupesnoop.pkg)
- [Windows Installer Download](https://github.com/jyecusch/dupesnoop/releases/latest/download/dupesnoop-amd64-installer.exe)
